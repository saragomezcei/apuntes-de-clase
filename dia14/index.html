Header

Definición: Cabecera de una sección o un documento
Semantic: Donde estará nuestro logo, nuestro título principal y la barra de navegación.

Nav

Definición: Elementos que nos permitirán movernos por el sitio web. Normalmente está dentro del header
Semantic: Identificar claramente las secciones que poseen enlaces de navegación y facilitan a los motores de búsqueda y usuarios moverse entre páginas del sitio web.

Main

Crea un contenedor para el contenido de la página

Section

Definición: Es un espacio genérico dentro de un documento (Div). Es donde se agrupan contenidos relacionados entre sí. Normalmente incluye un título. Una web se compone de secciones, cada una con su contenido y propósito.
Semantic:Ayuda a estructurar y organizar el contenido de la página en secciones lógicas que mejoran la comprensión del sitio.

Article

Definición: Define un contenido independiente que puede existir por sí mismo. Puede ser compartido y utilizado en otros sitios o secciones.
Ejemplos:

  • Un post de un blog
  • Un artículo de una revista
  • Comentario de un usuario
  • Card de un producto

Aside

Contenido relacionado INDIRECTAMENTE con el contenido de nuestro sitio
Ejemplos:

  • Publicidad
  • Biografía del autor
  • Enlaces relacionados
  • Información de perfil

∫

Footer

Pie de página, sitio o sección. Puede haber más de uno por página, pero no puede haber uno dentro de otro, o un footer dentro de un header. Ejemplos:

  • Contacto
  • Términos y condiciones
  • Sitemap
  • mapa

Otras secciones:

  • Figure: Definir un elemento autocontenido como una imagen, un gráfico, una tabla... Dentro de ella utilizamos la etiqueta "figcaption" que describe el contenido. (Es como un pie de foto, ES VISIBLE, no como el alt de las imágenes)
  • Mark: Resaltar contenido, suele aparecer en amarillo (en plan subrayador)
