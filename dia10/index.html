Position STATIC

Esta posición es la posición por default. No se aplica ninguna posición de position. Se muestra en el flujo normal del documento

Static

Position RELATIVE

Sí afecta el position (Top-Left-Right-Bottom).
Ocupa espacio en la página, aunque el elemento se mueva, el espacio en el que está (en teoria) no es ocupado por ningún otro elemento

Relative

Position ABSOLUTE

Sí afecta el position (Top-Left-Right-Bottom).
No ocupa espacio extra. Se posiciona respecto al padre más cercano que tenga una propiedad de position DISTINTA a STATIC

Absolute

Position FIXED

Sí afecta el position (Top-Left-Right-Bottom).
No ocupa espacio. Se posiciona con respecto a la ventana del navegador
Se mantiene fijo en la pantalla y no se mueve con el scroll.

Fixed

Position STICKY

Sí afecta el position (Top-Left-Right-Bottom).
Sí ocupa espacio. Se posiciona con respecto a la ventana del navegador
Se mantiene fijo en la pantalla y se queda fijo en pantalla hasta que el elemento padre se va.

Sticky
